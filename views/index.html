Youreuler

Thanks for checking Youreuler out. Do you have a URL? Is it too long? We have an api for that! Check it!

Send a POST request to /api/shorten with the following body: 

    {
      "original_url": "http://google.com/"
    }
And we'll send you back something like this: 
    {
      "original_url": "http://google.com/",
      "short_url": "http://youreuler.com/Itn0aD",
      "short_code": "Itn0aD",
      "slug": null
    }

That's all I got.
