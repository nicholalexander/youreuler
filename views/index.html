Do you have a URL? Is it too long? We have an api for that! Check it!

Send a POST request to /api/shorten with the following body: 

      {
        "original_url": "http://google.com/"
      }
And we'll send you back something like this: 
      {
        "original_url": "http://google.com/",
        "short_url": "http://www.youreuler.com/Itn0aD",
        "short_code": "Itn0aD"
      }

So you're one of those cgi, php sorts that likes to send parameters in the url? Fine. Post it to: 

 
      /api/shorten/?original_url=http://google.com/&slug='blonk'
and see if you don't get something nice back. And while we're at it, you can just GET it so you get something nice in the browser.

But god damn it, Jim, you're not here for short URLs, are you?! Of course not! You need long ones. We've got those too. Send the same json request to /api/enlengthen and you'll get yourself a real link. The other routes aren't available. For now.

A brief note on the naming. I met a man once who pronounced URL's not as it was meant to be pronounced, YOU-ARE-ELL, but rather as a word, thusly: YOU'RE UL. But as one word. Youreul. Thusly so we are named, in honor of that great and brave soul who flaunted his individuality in the face of both tradition and social expectations. We celebrate you, dear sir.
